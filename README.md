# 🚀 Chamados API

API REST desenvolvida com Java e Spring Boot para gerenciamento de chamados técnicos.

## 📋 Objetivo

Este projeto tem como objetivo simular um sistema de abertura e gerenciamento de chamados, aplicando boas práticas de desenvolvimento Back-end utilizando Java e Spring Boot.

## 🛠️ Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- Maven
- H2 Database
- Lombok
- Bean Validation

## 📂 Estrutura

```
controller
service
repository
entity
dto
exception
enums
```

## ⚙️ Funcionalidades

- Cadastro de chamados
- Listagem de chamados
- Consulta por protocolo
- Definição de prioridade
- Controle de status

## 🚧 Próximas melhorias

- PostgreSQL
- Docker
- Swagger/OpenAPI
- Testes unitários com JUnit e Mockito
- Autenticação JWT
- GitHub Actions (CI/CD)
